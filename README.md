## 2. 制作における工夫
プログラミングを学習しながら、以下の点に重点を置いて制作しました。

*   **クラス設計による整理**：シーン管理やStateパターンの導入など、コードが複雑になりすぎないよう設計を工夫しました。
*   **基礎技術の実装**：スマートポインタを用いたメモリ管理や、数学的な当たり判定ロジックの自作に挑戦しました。
*   **ビジュアル面での試行錯誤**：潜水艦の臨場感を出すため、シェーダー（HLSL）による水面の波や反射の表現、水中でのポストエフェクトの実装にも取り組みました。

## 3. 操作方法（簡易）
詳細は同封の解説書（PDF）をご確認ください。

| キー | 動作 |
| :--- | :--- |
| **W / A / S / D** | 移動（前後左右） |
| **Q / E** | 浮上 / 潜航（上下） |
| **マウス左クリック / Enter** | 魚雷発射 |

## 5. ビルド・実行方法
1.  ルートディレクトリにある `SubQuester.sln` を Visual Studio 2022 で開きます。
2.  ソリューション構成を **[Release]** または **[Debug]** 、プラットフォームを **[x64]** に設定します。
3.  ビルドを実行し、生成された実行ファイル（またはデバッグ開始）で起動してください。
    *   ※ 外部ライブラリは `libs` フォルダに同梱されており、相対パスで設定済みです。
    *   ※ 実行には `assimp-vc143-mt.dll` が実行ファイルと同じ場所に必要です。
